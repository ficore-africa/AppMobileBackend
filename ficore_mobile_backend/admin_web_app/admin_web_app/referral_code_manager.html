About Referral Codes

Assign custom referral codes to key partners and influencers. This allows them to share personalized codes (e.g., AUWAL2026) instead of auto-generated ones. Perfect for marketing campaigns and partner relationships.

  • Format: 3-20 characters, letters and numbers only
  • Use Case: Key partners, influencers, marketing campaigns
  • Tracking: All referrals are tracked in the Treasury Dashboard
